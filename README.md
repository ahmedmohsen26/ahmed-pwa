# يوميات أحمد — Personal Command Center 2026

## تشغيل الـ App

### على الكمبيوتر
افتح `index.html` في أي متصفح

### على الموبايل (Android)
1. افتح Chrome → `https://[username].github.io/ahmed-pwa`
2. القائمة (⋮) ← "Add to Home screen"
3. خلاص — icon على الشاشة الرئيسية!

### على الموبايل (iPhone)
1. افتح Safari → `https://[username].github.io/ahmed-pwa`
2. زرار المشاركة (□↑) ← "Add to Home Screen"

## رفع على GitHub Pages (مجاني)
1. github.com ← New repository ← اسمه `ahmed-pwa`
2. ارفع الملفات الأربعة
3. Settings ← Pages ← Source: main branch
4. بعد دقيقتين: `https://[username].github.io/ahmed-pwa`

## الملفات
- `index.html` — الـ app كاملاً
- `manifest.json` — إعدادات الـ PWA
- `sw.js` — Service Worker (offline support)
- `README.md` — هذا الملف
